W projekcie użyto 3 asercji:
- assertEquals - użyte do sprawdzania, czy ilość książek się zgadza po dodaniu/usunięciu,
- assertThrows - użyte do testowania, czy usunięcie książki, której nie ma, wywołuje wyjątek,
- assertTrue - użyte do sprawdzania, czy książka jest dostępna w księgarni.
